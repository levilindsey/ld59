This browser can't run ld59

ld59 uses Godot 4.6's multi-threaded web build, which requires SharedArrayBuffer and cross-origin isolation. Your current browser either blocks those features or hasn't shipped them.

Please open this page in one of:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

Safari currently doesn't implement the credentialless embedder policy that our threaded build needs, so the game won't load there until upstream Safari ships support.
